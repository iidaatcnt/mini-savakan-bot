# Mini-Savakan-Bot 仕様書 (SPEC)

## 1. プロジェクト概要
- **名称**: Mini-Savakan-Bot (ミニ・サバ缶・ボット)
- **目的**: ユーザーが Anthropic 社の CLI ツール「Claude Code」を PC に正常にインストールし、利用開始できるように支援する AI チャットボット。
- **ターゲット**: コマンドラインや環境構築に慣れていない初心者から、エラー解決を求める中級者まで。

## 2. システム構成
- **フロントエンド**: Next.js (App Router), React
- **スタイリング**: Tailwind CSS v4
- **アイコン/アニメーション**: Lucide React, Framer Motion
- **AI バックエンド**: Google Generative AI (Gemini API)
  - **使用モデル**: \`gemini-3-flash-preview\`
- **デプロイ環境**: Vercel

## 3. 主要機能
1. **ステップ進行型チャット UI**
   - 「環境診断」「インストール」「認証設定」「準備完了」の4ステップで構成。
   - **Step 1: 環境診断 (問診チェック)**: 6項目を厳格に確認し、導入不可なユーザーを早期選別。
   - **Step 2: インストール**: App StoreやCLI版の罠を回避し、公式サイトからアプリを導入・起動。
   - **Step 3: 認証・Proプラン**: 22ドル契約とSMS認証突破。重いXcodeを避けGit単体インストールを推奨。
   - **Step 4: 動作確認 (Read/Write)**: フォルダ読み書き権限の体験。ファイル不足時はGoogleドライブ連携を案内。
   - **Step 5: 活用と拡張**: Claude for Chrome拡張の設定、および公式「Ideas」内の「スキルス」活用案内。
   - ユーザーの進行状況（ステップ）に応じた文脈で AI が回答レベル（初心者向け or 応用）を調整。
2. **サバ管の知恵 (内部 RAG/FAQ 知識データ)**
   - 実際のオフ会等で発生した「あるあるエラー」の知見をシステムプロンプトにハードコード。
   - 今後、スケールアップの際は Google スプレッドシート API などに外部化予定。
3. **FAQ 検索機能（サイドバー）**
   - キーワード入力による FAQ 即時検索およびアコーディオン型での回答表示機能。
4. **執事ペルソナ（サバ缶様の筆頭執事・セバスチャンモデル）**
   - 極めて丁寧だが慇懃無礼で毒舌なサポート役。初心者を優雅に導きつつ、古いやり方や情報商材には厳しいスタンスをとる。

## 4. セキュリティ・ガードレール
- **プロンプトインジェクション防止**: 役割変更やシステム指示の無視要求を全て拒否。
- **話題制限**: IT、Claude Code 関連以外の質問（政治、エンタメ等）への回答拒否。
- **個人情報保護**: プロンプトに埋め込まれた設計者（サバ缶）の個人情報を秘匿。
